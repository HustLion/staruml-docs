==============
Basic Concepts
==============

StarUML is a tool for building software models.

Elements
========

Element is building block constitutes a software model.

There are three kinds of Elements in StarUML.

* Model Element
* View Element
* Diagram

StarUML makes a clear conceptual distinction between Models, views and diagrams. A Model is an element that contains information for a software model. A View is a visual expression of the information contained in a model, and a Diagram is a collection of view elements that represent the userâ€™s specific design thoughts.
